📌 QR Code Generator
A simple and responsive QR Code Generator web application that allows users to generate QR codes from text or URLs and download them instantly.

🚀 Features

🔤 Convert text or URLs into QR codes
📏 Multiple size options (100x100 to 1000x1000)
📥 Download QR code as an image
⚡ Instant QR code generation
📱 Responsive design for all devices

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & layout
JavaScript (Vanilla JS) – Functionality
QRCode.js Library – QR generation
